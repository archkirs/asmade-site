About

The product is not the code. The product is the reviewable record behind it.

AI-assisted work can produce many materials: final files, AI chats, drafts, screenshots, source notes, references, versions, and metadata. Sending all of that at once can be too much for a reviewer.

This landing tests a simple mechanism: one short code or link that points to a structured process record. The reviewer sees the summary first, then understands what evidence exists and what cannot be concluded.

FAQ

A code helps navigation. It does not replace review.

This pilot tests whether a code or link makes process evidence easier to share. It does not create a public database or automated verification system yet.

The pilot does not independently determine legal ownership, copyright status, publication rights, academic acceptance, platform compliance, or originality.
